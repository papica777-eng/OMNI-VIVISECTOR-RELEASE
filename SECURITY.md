# Security Policy

## Supported Versions

AETERNA provides active security patching and support for the following OMNI-VIVISECTOR versions:

| Version | Enterprise Support Status | SLA Response Time |
| ------- | ------------------------- | ----------------- |
| > 8.x   | ✅ Fully Supported         | < 2 Hours         |
| 7.x     | ⚠️ Critical Patches Only   | < 24 Hours        |
| < 7.0   | ❌ End of Life             | None              |

## Vulnerability Reporting

If you have discovered a vulnerability in the OMNI-VIVISECTOR engine, the CLI, or the MCP Server, please **DO NOT** open a public issue.

To ensure the highest level of Operational Security (OpSec):
1. **Email:** Send a PGP-encrypted report to `security@aeterna.website`.
2. **Details:** Include the exact version, environment details, and a reproducible Proof-of-Concept.
3. **Bounty:** Critical bypasses or Sandbox Escape findings are eligible for up to a **$100,000 USD** bug bounty.

### Response Timeline
- Acknowledgment: within **12 hours**
- Verification: within **48 hours**
- Patch formulation and deployment: within **72 hours**

All Enterprise (`VORTEX_EMPIRE`) clients will receive an automated zero-day hotfix via the secure over-the-air (OTA) patching mechanism.
